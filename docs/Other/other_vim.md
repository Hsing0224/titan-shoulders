# Vim

內建在 Unix 的終端機文字編輯器

## 開啟

```shell
$ vim 檔案.副檔名
```

## 模式

- 一般模式：進入 Vim 的預設狀態，用來移入游標、刪除、複製。
- 輸入模式：可編輯文件內的文字。
- 命令列模式：存檔、退出、設定環境。

:::note
不管哪個模式下，按下 `Esc` 可以回到一般模式
:::

### 一般模式

#### 切換至輸入模式

- `i`: 切換成輸入模式
- `a`: 在游標後面並進入輸入模式
- `A`: 跳至該行結尾並切換到輸入模式
- `o`: 在游標位置下方新增一行並切換到輸入模式

#### 移動

- `h` / `j` / `k` / `l`: 左下上右，前面輸入數字可以移動幾次。例： `20l` 往右 20 個字元
- `w`: 往後移動一個單字/符號
- `e`: 往後移動一個單字/符號的結尾
- `b`: 往前移動一個單字/符號
- `gg`: 到第一行
- `G`: 到最後一行
- `f`: 搜尋該行。例： `fn` 會搜尋最近一個 `n` 的單字，繼續輸入 `fn` 可搜尋下一個 `n`
- `/`: `/{關鍵字}` 可以搜尋，按下 `n` 可以跳下一個符合的結果，按 `N` 可以跳到上一個符合的結果

#### 編輯

- `x`: 刪除游標單一字元 (同delete)
- `dd`: 剪下整行
- `yy`: 複製整行
- `p`: 在游標的下一行貼上複製或剪下的內容
- `u`: 還原上一步

### 命令列模式

在一般模式下，輸入 `:` 會進入命令列模式

- `:{num}`: `:20` 移動至第 20 行。
- `:q`: 退出，無修改過檔案，直接退出。
- `:wq`: 存檔退出。
- `:q!`: 強制退出、不存檔。
- `:w`: 單純存檔。
