# Cron 表達式

Linux 或 Unix 設定時間用的表達式。<br />
用於排程上的時間設定來觸發任務<br />
通常會有 5 到 6 位數來設定時間

## 寫法

### 5 位數( Linux 預設)

```
  * * * * *
# | | | | |
# | | | | 星期幾 (0–6) (週日到週六，有些系統 7 也代表週日；可填英文，例： MON)
# | | | 月份 (1–12) (可填英文，例： JAN)
# | | 日 (1–31)
# | 小時 (0–23)
# 分鐘 (0–59)
```

### 6 位數

有些軟體服務才有支援 6 位數的設定

```
  * * * * * *
# | | | | | |
# | | | | | 星期幾
# | | | | 月
# | | | 日
# | | 時
# | 分
# 秒 (0–59)
```

:::danger

#### 關於秒的設定

如果寫六位數時，秒的位置寫 `*`，則是**每秒**觸發的意思。<br />
一般都會填入 `0`，不然就會在符合條件的那一分鐘裡的每秒都會被觸發。

```
* 0 9 * * 1-5 # 在每週一至五的早上 9:00 的一分鐘接會每秒觸發
```

:::

## 符號

- `*`: 星號代表「每」，例如在**分**寫 `*`，代表每分鐘。
- `,`: 逗號代表「和」，例如在**時**的位置寫入 `9,18`，代表早上 9 點和下午 6 點。
- `-`: 連字號代表「範圍」，例如在**週**位置寫入 `1-5`，代表週一到週五。
- `/`: 斜線代表「每隔」，例如在**月**位置寫入 `*/2`，代表每隔兩個月。

## 常見範例

| 表達式        | 說明                           |
| :------------ | :----------------------------- |
| `* * * * *`   | 每分鐘執行一次                 |
| `*/5 * * * *` | 每 5 分鐘執行一次              |
| `30 16 * * *` | 每天 16:30 執行一次            |
| `0 9 * * 1-5` | 每週一至週五早上 9:00 執行一次 |
