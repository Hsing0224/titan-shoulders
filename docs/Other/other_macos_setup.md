# MacOS 開發環境建置

## Homebrew

Homebrew 是 MacOS/Linux 系統的套件管理工具，可至[官網](https://brew.sh/)透過步驟安裝

```bash
# /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

```bash
$ brew install [要安裝的套件]
```

### node

JavaScript 執行環境

### git

版本控制套件

### uv

Python 套件管理工具

### ripgrep

純文字搜尋工具
